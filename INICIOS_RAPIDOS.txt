🎨 PERSONALIZACIÓN RÁPIDA

📝 CAMBIAR EL MENSAJE:
   ├─ Abre: index.html
   ├─ Busca: class="message-text"
   └─ Edita el texto entre las etiquetas <p>

📸 AGREGAR FOTO:
   ├─ Una vez abierta la carta, haz clic en la foto
   ├─ Selecciona una imagen de tu dispositivo
   └─ ¡Se mostrará automáticamente!

✍️ CAMBIAR LA FIRMA:
   ├─ Abre: index.html
   ├─ Busca: class="signature-name"
   └─ Cambia "Tu Admirador" por tu nombre

🎨 CAMBIAR TEMA DE COLOR:
   ├─ Abre: Jss/config.js
   ├─ En CONFIG.theme, cambia:
   │  ✓ rose (rosa, por defecto)
   │  ✓ purple (púrpura)
   │  ✓ blue (azul)
   │  ✓ gold (dorado)
   │  ✓ red (rojo)
   └─ Recarga la página para ver el cambio

🌸 CAMBIAR LAS FLORES:
   ├─ Abre: index.html
   ├─ Busca: <div class="flower..."
   └─ Cambia los emojis: 🌸 🌺 🌼 🌻 🌷 💐 🥀

1. Para mejor experiencia, usa un navegador moderno
   (Chrome, Firefox, Edge o Safari recomendados)

2. Las flores comienzan a caer poco después de cargar
   Espera unos segundos para verlas

3. Usa fotos en formato cuadrado para mejor resultado
   Formatos soportados: JPG, PNG, GIF, WebP

4. El sitio funciona completamente sin conexión a internet
   Una vez cargado, puedes desconectarte

5. El contenido de la foto se guarda solo en tu sesión actual
   No se guarda permanentemente o en servidores

6. Edita el archivo README.md para más opciones avanzadas

🆘 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❓ Las flores no están cayendo
   → Espera unos segundos después de cargar
   → Actualiza la página (F5)

❓ La foto no carga
   → Verifica que el formato sea JPG, PNG, GIF o WebP
   → Asegúrate que la imagen no sea muy grande
   → Intenta nuevamente con otra foto

❓ Las animaciones están lentas
   → Cierra otras pestañas del navegador
   → Cierra otras aplicaciones
   → Reinicia el navegador

❓ El sonido no funciona
   → Es normal en algunos navegadores (privacidad)
   → La carta funciona perfectamente sin sonido
   → Esto no afecta ninguna otra funcionalidad

❓ El mensaje no se centra bien
   → Verifica que el contenido no sea demasiado largo
   → Si necesitas texto más largo, usa saltos de línea (<br>)
